# Make Internet UI Great Again (MIUIGA)

MIUIGA is a web framework dedicated to reclaiming the internet from the era of "clean," "flat," and "minimalist" design. It embraces the unapologetically quirky, modular, and tactile aesthetics of the Web 1.0 era, while providing an optional path into Web 2.0.

## Features

- **Retro Aesthetic**: 3D bevels, high-contrast textures, and animated GIFs.
- **Classic Layouts**: "Frameset" column layouts without using legacy `<frameset>` tags.
- **Interactive Widgets**: Visitor counters, media players, guestbooks, and an algorithmic webring router.
- **Toggleable Modern Mode**: Switch the theme to a late 2000s / early 2010s "Web 2.0 Glossy Portal" style (inspired by the GameTrailers.com layout) featuring metallic steel-blue gradients, glossy glass shine buttons, rounded corners, sleek custom scrollbars, and bubble-drop cursor trails.
- **AOL Buddy List**: Online/offline buddy widgets with mock IM and Chat triggers.
- **RPG Revelation Pathway & Avatars**: Monospace breadcrumb paths and pixel badge avatars next to blog post headings.
- **Strict 216 Color Palette**: Enforced CSS properties restricting classic retro colors to the mathematical web-safe RGB grid.
- **Dithered Scrollbars**: Microscopic 2x2 SVG checkerboards rendering non-anti-aliased tracks.
- **Global Pixel Preservation**: Enforced nearest-neighbor scaling resets for retro bitmap fidelity.
- **High-Performance Trails**: Full-screen Canvas cursor overlays animating at 60FPS.
- **Standardized IAB Banners**: Mapped to 1990s Interactive Advertising Bureau ad specs (468x60, 392x72, 234x60, 120x240).
- **Spacer GIF Emulation**: A Custom HTML element `<spacer-gif>` mimicking historical layout spacers.
- **Draggable OS Windows**: Windows 95/98 modal popup dialogs with drag-and-drop mechanics.
- **Table Grid Emulation**: Modern CSS Grid table structures with a toggleable `border="1"` dotted outline debug view.
- **Typography Expansions**: Comic Sans MS and Verdana stacks with disabled subpixel anti-aliasing.
- **Zero Dependencies**: Pure Vanilla JS and CSS.

## Getting Started

Simply open `index.html` in your browser to experience the MIUIGA framework showcase (a retro personal blog: "The Webmaster's Cyber-Log"). Click the navigation buttons to toggle table borders, change background tiles (including the new purple **NEBULA** theme), open OS warning dialogs, or toggle **MODERN MODE** to step from the late 90s into the glossy early 2010s portal layout.
