# Make Internet UI Great Again (MIUIGA)

MIUIGA is a web framework dedicated to reclaiming the internet from the era of "clean," "flat," and "minimalist" design. It embraces the unapologetically quirky, modular, and tactile aesthetics of the Web 1.0 era.

## Features

- **Retro Aesthetic**: 3D bevels, high-contrast textures, and animated GIFs.
- **Classic Layouts**: "Frameset" look without the actual `<frameset>` tag.
- **Interactive Widgets**: Visitor counters, media players, and guestbooks.
- **Zero Dependencies**: Pure Vanilla JS and CSS.

## Getting Started

Simply open `index.html` in your browser to experience the MIUIGA framework.
